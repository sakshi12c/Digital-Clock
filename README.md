Features

Real-time clock display that updates every second
Toggle between 12-hour (AM/PM) and 24-hour formats
Large, easy-to-read display
Clean design with visual feedback

Files

DigitalClock.html - Main page structure
DigitalClock.js - Clock logic and time formatting
DigitalClock.css - Styling and layout

How to Use

Open DigitalClock.html in a web browser
The clock displays current time automatically
Click "12 Hour" for 12-hour format with AM/PM
Click "24 Hour" for 24-hour military format

Installation

Download all three files
Place them in the same folder
Open DigitalClock.html in any browser

Customization
Change Clock Size
Edit in DigitalClock.css:
css.clock-display {
    font-size: 72px;  /* Adjust this value */
}
Change Colors
Edit in DigitalClock.css:
css.clock-display {
    color: #333;  /* Clock color */
}
Browser Compatibility
Works on Chrome, Firefox, Safari, Edge, and Opera.
